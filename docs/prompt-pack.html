PROMPT PACK

Copy-and-paste prompts for every stage of the Curator Cards workflow.

PHASE 1 — Exploration

Use these prompts to survey the territory before drafting.

Concept Map Prompt:

Create a concept map showing how Bauhaus (1919–1933), Swiss Style (1950s–1970s), and New Wave Typography (1970s–1980s) relate to each other. Include key figures and core principles.

Controlled Vocabulary Prompt:

Generate 20 domain-specific terms related to Bauhaus, Swiss Style, and New Wave Typography. Define each in one sentence.

Confusions Prompt:

Explain common confusions between Bauhaus vs Art Deco, Swiss Style vs Minimalism, and New Wave vs Grunge Typography.

PHASE 2 — Production

Draft Prompt:

Write three museum-label style Curator Cards (80–120 words each) using only controlled vocabulary. Include 5 tags, 2 example links, a confusion sentence, and an accountability note.

Self-Audit Prompt:

Audit the three Curator Cards for word count, tag count, link count, confusion sentence, and accountability note.

PHASE 3 — Expert Critique

Critique Prompt:

Review my Curator Card about [movement] as if you were [expert name]. Critique accuracy and vocabulary. Flag anything uncertain as VERIFY.
