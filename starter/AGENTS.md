Controlled Vocabulary Table

Vorkurs — Foundation course at the Bauhaus teaching materials, color theory, and formal experimentation.

Form follows function — Design principle that form should derive from purpose and utility.

Geometric abstraction — Use of simplified geometric forms instead of representational imagery.

Modular composition — Design built from repeatable structural units.

Functionalism — Design approach prioritizing utility over ornament.

Grid system — Underlying structural framework organizing text and image.

Asymmetrical layout — Balanced composition without mirrored symmetry.

Sans-serif typography — Typefaces without decorative finishing strokes.

Objective typography — Swiss principle prioritizing clarity and neutrality.

Typographic hierarchy — System of organizing type by size, weight, and spacing to guide reading.

Baseline grid — Horizontal alignment system ensuring consistent text flow.

Photomontage — Composite imagery created by assembling photographs.

Expressive typography — Typography used as visual form, not just text.

Typographic layering — Overlapping text elements to create spatial tension.

Grid disruption — Intentional breaking of grid alignment rules.
